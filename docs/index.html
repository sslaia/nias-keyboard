Key capabilities

Everything you need, built-in

✨

Dual-Language Autocomplete

Features smart word completion for both Indonesian and Nias (Li Niha). Auto-learns new vocabulary as you type to make predictions more personalized.

💡 Note: Toggle the Wiki codes row off in order to display dictionary word completions.

📝

Wiki Codes Toggle

A specialized quick-access row designed for Wikipedia volunteers. Input syntax shortcuts like [[, ]], {{, and }} instantly. Toggle the bar on or off at the left corner of the row.

⌨️

Native Special Characters

Type native characters directly. Press keys long-press for Nias letters and international symbols: ŵ (w), ö (o), ä (a), ü (u), € (e), £ (l), and more.

Setup Guide

How to enable Nias Keyboard

1

Enable in Settings

Open the app, tap Activate Keyboard and toggle Nias Keyboard on in your system settings.

2

Switch Input Method

Focus any text field, tap the small keyboard icon at the bottom right corner of the screen, and select Nias Keyboard.

3

Start Typing

Switch between Nias and Indonesian using the globe icon (🌐). Use the Wiki toggle to access editing codes instantly!
